# BrightcoveThis is a repo for managing machine readable artifacts for the [Brightcove APIs](http://brightcove.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Brightcove APIs](http://brightcove.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://brightcove.com)- [Developer](http://brightcove.com)- [Twitter](https://twitter.com/brightcove)- [GitHub](https://github.com/brightcove)The [APIs.json](https://github.com/api-evangelist/brightcove/blob/master/apis.json) index provides links to all the relevant links for the Brightcove APIs, including the OpenAPI and Postman if they are present.
